### 密钥生成
```
ssh-keygen -t rsa
ssh-keygen -t rsa -C "hello@hello.com"
ssh-keygen -t rsa -C "hello@hello.com" -f id_key_name
```

### 加载私钥
```
ssh-add id_key_name
```

### 卸载私钥
```
# 卸载所有私钥
ssh-add -D
```

### 列出所有已加载密钥
```
ssh-add -l
```

### 多个密钥管理

#### 1. 创建配置文件
```
touch ~/.ssh/config
```

#### 2. 编辑配置
```
# vim ~/.ssh/config
Host github.com
   HostName github.com
   User git
   IdentityFile ~/.ssh/id_rsa_name1

Host gitlab.com 
   HostName gitlab.com
   User git
   IdentityFile ~/.ssh/id_rsa_name2
```

### 利用别名指令
```
alias ss1='ssh-add -D && ssh-add ~/.ssh/id_123'
```
