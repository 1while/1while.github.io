# 强制深色模式设置
1. chrome访问`chrome://flags/#enable-force-dark`
2. 将`Auto Dark Mode for Web Contents`设置为enabled
![dark mode](img/chrome_dark_mode.png)
3. 访问`chrome://settings/appearance`, 主题背景选GTK+
