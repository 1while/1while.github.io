# 安装
```bash
# for fedora
sudo dnf install emacs

# for macos
brew install emacs
```

# 怎么退出emacs
```bash
# 先后输入组合键Ctrl+x, Ctrl+c
C-x C-c

# 关闭当前缓冲区
C-x k Enter
```
