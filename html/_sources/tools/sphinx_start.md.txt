
# sphinx-quickstart

sphinx-quickstart

make html

## 自动编译 - 安装
pip install sphinx-autobuild

sphinx-autobuild source build/html

pip install furo

## markdown 支持

pip install sphinx_markdown_tables


pip install myst-parser
