ohmyzsh 安装p10k主题

1. 下载仓库 

1

git clone --depth=1 git@github.com:romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

2. 设置主题 

1
2

$ vi ~/.zshrc
ZSH_THEME="powerlevel10k/powerlevel10k"

3. 重新加载配置生效 

1

$ source ~/.zshrc

4. 加载完配置后默认会自动执行一次个性配置引导 

1
2

# 手动执行引导
$ p10k configure
